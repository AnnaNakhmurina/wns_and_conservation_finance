<h1 align="center">Support Code for <br> 
  "The Fiscal Impact of Biodiversity Loss and a Pathway for Conservation Finance"
</h1>

  ## Anya Nakhmurina, Dale T. Manning, Eli P. Fenichel

This code enables replication of results in "The Fiscal Impact of Biodiversity Loss and a Pathway for Conservation Finance," published in _Science_ XXXX.

All data sources for the project are available to research, though some require licenses.  
* The white noise syndrome data come from https://www.whitenosesyndrome.org/.  
* The municipal bond data come from Mergent https://bv.mergent.com/view/scripts/common/index.php, these data were accessed with an academic license.
* The socioeconomic data come from the Government Finance Database https://my.willamette.edu/site/mba/public-datasets.

We also use:  
* The weather data from http://www.prism.oregonstate.edu
* The property tax data from CoreLogic (now Cotality) https://www.cotality.com/products/property-tax-data-products, these data were accessed with an academic license. We are grateful to Cameron LaPoint for organizing these data for us.
* The bat species richness data from Frank (2024).
* The rural county classification data from the US Department of Agriculture.  
* The probability of bat detection data from https://www.nabatmonitoring.org.
* The secondary bond trading data from https://wrds-www.wharton.upenn.edu/wrds/ds/compd/fundq/index.cfm.
