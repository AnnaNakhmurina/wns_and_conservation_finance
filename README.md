# Support Code for "The Fiscal Impact of Biodiversity Loss and a Pathway for Conservation Finance"
## Anya Nakhmurina, Dale T. Manning, Eli P. Fenichel

This code enables replication of results in "The Fiscal Impact of Biodiversity Loss and a Pathway for Conservation Finance" publish in XXXX

All data source for the project are available to research, though some require licenses.  
* The white noise syndrome data come from whitenosesyndrome.org.  
* The municipal bond data come from Mergent https://bv.mergent.com/view/scripts/common/index.php, these data were accessed with an academic license.  

We also use:  
* weather data from http://www.prism.oregonstate.edu
