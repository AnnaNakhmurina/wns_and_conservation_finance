<h1 align="center">Support Code for <br> 
  "The Fiscal Impact of Biodiversity Loss and a Pathway for Conservation Finance"
</h1>

  ## Anya Nakhmurina, Dale T. Manning, Eli P. Fenichel

This code enables replication of results in "The Fiscal Impact of Biodiversity Loss and a Pathway for Conservation Finance," published in _Science_ on March 12, 2026.

## Data Sources
All data sources are available to researchers, though some require licenses.

### Primary Data
- **White-nose Syndrome occurence data**  from [White-Nose Syndrome Response Team](https://www.whitenosesyndrome.org/)  
- **Municipal bond issuance data** from [Mergent](https://bv.mergent.com/view/scripts/common/index.php) (accessed with an academic license)
- **The socioeconomic data** from the [Government Finance Database](https://my.willamette.edu/site/mba/public-datasets)

### Additional Data 
- **Weather data** from [PRISM Climate Group](http://www.prism.oregonstate.edu)
- **Property tax data** from [CoreLogic (now Cotality)](https://www.cotality.com/products/property-tax-data-products) (accessed with an academic license). We are grateful to Cameron LaPoint for organizing these data for us.
- **Bat species richness data** from Frank (2024)
- **Rural county classification data** from the [U.S. Department of Agriculture](https://www.ers.usda.gov/data-products/rural-urban-continuum-codes/)
- **Bat detection probability** from [North American Bat Monitoring Program (NABat)](https://www.nabatmonitoring.org/)
- **Secondary bond trading data** from [WRDS](https://wrds-www.wharton.upenn.edu/)
